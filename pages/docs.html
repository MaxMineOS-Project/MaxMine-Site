Документация API для разработчиков

В нашей командной оболочке есть API для создания приложений. Для начала, напишите приложение на Python. Оно может включать любое количество файлов с любым именем, но должно также содержать файл main.py, в котором есть функция main, принимающая аргументы:

  • только argv (аргументы командной строки, включая имя приложения)
  • или argv и abspath (путь установки командной оболочки)
  • или argv, abspath и current_user (имя текущего пользователя)

Типы аргументов:

  • argv: list[str]
  • abspath: str
  • current_user: str

После этого вы можете упаковать своё приложение в zip-архив с именем <имяприложения>.mos. Затем отправьте его с описанием на почту maximdev@internet.ru. Вы будете уведомлены, если ваше приложение пройдёт модерацию.

Our command shell provides an API for creating applications. To get started, write an application in Python. It can include any number of files with any names, but must also contain a main.py file with a main function that accepts arguments:

  • Only argv (command-line arguments including the app name)
  • Or argv and abspath (shell installation path)
  • Or argv, abspath, and current_user (current username)

Argument types:

  • argv: list[str]
  • abspath: str
  • current_user: str

Then you can package your app into a zip archive named <appname>.mos. Send it with a description to maximdev@internet.ru. You will be notified if your app passes moderation.
